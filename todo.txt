## üéØ SELECTION CRITERIA FOR NEXT TOOL
======================================
When choosing what to build next, consider:

**High Priority Factors:**
- Reuses existing patterns successfully
- Provides high user value (reference or daily use)
- Complements existing tools
- Manageable data collection effort
- No external dependencies required

**Medium Priority:**
- Introduces one new technical pattern
- Moderate complexity calculations
- Requires research/data gathering
- Could use optional API

**Lower Priority (Save for Later):**
- Requires backend/database
- Needs multiple new technologies
- Complex astronomical calculations
- Community features requiring moderation
